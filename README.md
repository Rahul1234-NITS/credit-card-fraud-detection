# Credit Card Fraud Detection

## About
Built a machine learning model to detect fraudulent credit card 
transactions using Random Forest Classifier.

## Dataset
Download from Kaggle: 
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SMOTE

## Results
- F1-Score: 1.00 (100%)
- ROC-AUC Score: 0.9999
- Only 9 transactions misclassified out of 113,726

## Steps
1. Exploratory Data Analysis (EDA)
2. Feature Scaling
3. SMOTE for class imbalance
4. Random Forest Model
5. Evaluation
